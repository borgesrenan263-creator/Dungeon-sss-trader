# Dungeon SSS Trader

Galaxy Isekai Protocol MMORPG Economy Engine.

Core Gameplay Loop:

Matar monstros
↓
Dropar cristais de mana
↓
Converter em ouro
↓
Refinar equipamentos
↓
Vender itens
↓
Subir no ranking global

Sistema econômico:

Ouro
Obsidiana (limitado a 5.000.000)
USDT para comércio P2P

Principais mecânicas:

• Farm puro
• Upgrade arriscado
• Boss global
• Ranking Top 10
• Mercado entre jogadores
• Economia escassa

Projeto em desenvolvimento.
